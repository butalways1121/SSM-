# SSM-StudentsManeger
SSM框架的学生信息管理系统
环境
  系统环境：Windows7
  开发工具：Eclipse j2ee 2019
  Java版本：JDK 11.0.2
  服务器：tomcat 7.0
  数据库：MySQL 8.0
  数据库工具：Navicat for mysql 12
  系统采用技术： SSM+Jsp+EasyUI+jQuery+Ajax+面向接口编程
  
系统功能：
  系统分管理员和学生两个角色：
    管理员可实现对地区信息、学校信息、年级信息、班级信息、学生信息进行添加、修改、删除、查询等功能，
    学生可以对地区信息、学校信息、年级信息及班级信息进行查看，但在学生信息页，只能对自己的相关信息查看和修改。
